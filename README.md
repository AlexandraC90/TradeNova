TradeNova is a desktop trading dashboard built with JavaFX.  
The project focuses on creating a modern fintech-style interface using a dark corporate theme and custom CSS styling.

Overview

The application simulates the layout and visual structure of a professional trading platform.  
It demonstrates UI composition, reusable components, and JavaFX scene management.

Features

- Dark themed trading dashboard
- Portfolio analytics view
- Markets view
- Settings panel
- Custom styled components
- Maven based project structure

Technology Stack

- Java
- JavaFX
- Maven
- CSS

Running the Application

mvn javafx:run
